# 键盘快捷键 & 音频可视化功能

## ⌨️ 键盘快捷键

### 播放控制

- `Space` - 播放/暂停
- `→` (右箭头) - 下一首
- `←` (左箭头) - 上一首/重新播放

### 音量控制

- `↑` (上箭头) - 音量增加 10%
- `↓` (下箭头) - 音量减少 10%
- `M` - 静音/取消静音

### 播放模式

- `L` - 切换播放模式（顺序/循环/随机）

### 注意事项

- 在输入框中输入时，快捷键不会触发
- 所有快捷键都会阻止默认行为

## 🎵 音频可视化

### 功能特点

- **实时频谱分析**：使用 Web Audio API 分析音频
- **8 条频谱柱**：显示不同频率范围
- **动态高度**：根据音频强度实时变化
- **待机动画**：未播放时显示脉冲动画

### 显示位置

- 全局播放器顶部
- 播放控制按钮上方

### 技术实现

- Web Audio API
- AnalyserNode
- requestAnimationFrame
- 实时数据更新
