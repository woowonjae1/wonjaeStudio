# Woowonjae's Music Blog

这是一个使用现代化技术栈构建的音乐博客网站。

## 技术栈

- Next.js 14 (React Framework)
- TypeScript
- Tailwind CSS
- Three.js
- Framer Motion (动画效果)
- React Three Fiber (Three.js React 渲染器)

## 功能特点

- 响应式设计
- 3D 专辑展示
- 音频播放器
- 博客文章展示
- 滚动动画效果
- SEO 优化

## 项目结构

项目采用模块化设计，主要分为以下几个部分：
- 布局组件 (Header, Footer, Navigation)
- 页面部分 (About, Blog, Music, Albums)
- 专辑展示 (3D 渲染, 播放控制)
- 音频播放器
